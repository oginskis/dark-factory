# Product Taxonomy

This is the canonical taxonomy of tangible goods categories. Every entry describes a physical product or class of physical products. All other skills that need product classification read from this file.

This file is edited manually — the `product-taxonomy` skill reads from it but does not modify it. All other skills are read-only.
